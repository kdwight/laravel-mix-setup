# Laravel Mix (v6.0) Initial Setup
## Setup
- Clone this repo
- Update `.env` variables
- `npm install`
- `npx mix`

<br />

## npm terminal scripts

| *npm*               | *npx* (laravel-mix v6.0)  |
|--------------------|----------------------------|
| `npm run dev`      | **`npx mix`**              |
| `npm run watch`    | **`npx mix watch`**        |
| `npm run prod`     | **`npx mix --production`** |
